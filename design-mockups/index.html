Nine directions. 09

Nine distinct design philosophies for the same sheet. Same sample data across all of them (6 creatives spanning Massive Winner → Loser) so you can compare fairly. Round 1 explored foundational options. Round 2 incorporates current industry references — Anthropic Claude, Vercel Geist, PostHog data-viz, Raycast, and Swiss editorial.

How to choose

  • Density — Linear / Geist / Terminal / Data-viz are tight; Claude / Editorial / Swiss give breathing room
  • Mode — Dark: Linear, Terminal, Geist, Raycast · Light: Editorial, Stripe, Claude, Data-viz, Swiss
  • Accent philosophy — One color (Geist, Swiss), warm (Claude, Editorial), colorful (Raycast, Stripe)
  • Hero element — Typography (Claude, Swiss) vs data (Data-viz, Terminal) vs hierarchy (Linear, Stripe)
  • Navigation — Sidebar pattern in 05-Claude and 06-Geist (matches 2026 Vercel redesign standard)
  • Pick hybrid — Most real choices combine directions (e.g. Claude sidebar + Data-viz charts + Geist table)
